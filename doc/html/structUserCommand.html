Detailed Description

This structure represents user command. User command can execute any task. It is usually not present on GPU. The closest GPU command would be compute shader. When the GPU encounter this command, it will execute user callback and give the command userData. If the callback is nullptr, the GPU ignores it. Student should implement correct behaviour of this command.
