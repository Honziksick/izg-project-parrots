Set Backface Culling \doxylink{structCommand}{Command} can enable/disable backface culling. This command can enable/disable backface culling. Also, it can specify, what is backfacing face of a triangle. Backface culling disable rasterization of triangles that are backfacing a camera. The backfacingness is determined by the order of vertices -\/ clockwise or counter clockwise.
