Per Fragment operace jsou operace, které se provádí nad fragmenty, které vyprodukovala rasterizace. Jsou rozděleny na tři části\+:
\begin{DoxyItemize}
\item Brzké Per Fragment Operace
\item Fragment Shader
\item Pozdní Per Fragment Operace
\end{DoxyItemize}



Brzké Per Fragment Operace jsou složeny z\+:
\begin{DoxyItemize}
\item Stencil Testu
\item Depth Testu
\end{DoxyItemize}

Pozdní Per Fragment Operace jsou složeny z\+:
\begin{DoxyItemize}
\item Operace Discard
\item Zápisu stencilu
\item Zápisu hloubky
\item Zápisu barvy
\end{DoxyItemize}
